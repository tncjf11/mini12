# 🍽️ CookMate - 레시피 공유 앱

React와 Firebase를 기반으로 한 레시피 공유 플랫폼입니다. 사용자들이 자신만의 레시피를 공유하고, 다른 사람들의 레시피를 찾아보며, 마음에 드는 레시피를 찜할 수 있는 커뮤니티 앱입니다.

## ✨ 주요 기능

### 🎨 **통일된 디자인 시스템**
- **헤더 컴포넌트 통합** - 모든 페이지에 일관된 상단 헤더
- **로고 중앙 정렬** - 로그인 상태와 관계없이 완벽한 중앙 배치
- **반응형 네비게이션** - 통일된 하단 네비게이션 바
- **일관된 UI/UX** - 모든 페이지에서 동일한 디자인 패턴

### 🔐 **인증 시스템**
- **이메일 회원가입/로그인** - Firebase Auth 기반
- **구글 소셜 로그인** - 간편한 소셜 로그인
- **실시간 인증 상태 관리** - 자동 로그인 상태 유지
- **보안 규칙** - Firestore 보안 규칙 적용

### 🏠 **메인 홈페이지**
- **베스트 메뉴 추천** - 인기 레시피 노출
- **카테고리별 탐색** - 한식, 양식, 일식, 중식
- **검색 바로가기** - 원클릭 검색 페이지 이동
- **스크롤 배너** - "LET'S GET COOKING" 애니메이션

### 🔍 **검색 및 탐색**
- **실시간 검색** - 제목, 설명, 작성자 검색
- **카테고리 필터링** - 8개 카테고리 지원
- **URL 파라미터 지원** - 카테고리 직접 링크
- **검색 결과 카운트** - 검색된 레시피 개수 표시
- **로딩 및 오류 상태** - 사용자 친화적 상태 관리

### 📝 **레시피 작성**
- **상세 레시피 입력** - 제목, 재료, 설명, 요리과정
- **단계별 요리과정** - 동적 단계 추가/삭제
- **카테고리 및 난이도** - 8개 카테고리, 3단계 난이도
- **실시간 유효성 검사** - 입력 오류 즉시 표시
- **Firebase 연동** - 실제 데이터베이스 저장

### 📖 **레시피 상세 보기**
- **완전한 레시피 정보** - 재료, 과정, 난이도, 시간
- **찜하기 기능** - 실시간 찜 상태 동기화
- **작성자 정보** - 프로필 및 닉네임 표시
- **좋아요/댓글 수** - 인기도 표시

### ❤️ **찜 기능**
- **실시간 찜 관리** - Firestore 실시간 동기화
- **찜 목록 보기** - 개인 찜 레시피 모음
- **빠른 접근** - 찜 페이지에서 레시피 바로 이동
- **로그인 유도** - 비로그인 시 적절한 안내

### 👤 **마이페이지**
- **프로필 관리** - 사용자 정보 표시
- **내 레시피/찜 탭** - 작성한 레시피와 찜한 레시피 분리
- **레시피 그리드** - 깔끔한 2열 그리드 레이아웃
- **로그아웃 기능** - 안전한 로그아웃

### 📱 **네비게이션**
- **통일된 상단 헤더** - 모든 페이지 공통 헤더
- **통일된 하단바** - 모든 페이지 공통 네비게이션
- **현재 페이지 표시** - 활성 상태 하이라이트
- **아이콘 + 텍스트** - 직관적인 네비게이션

## 🛠️ 기술 스택

### **Frontend**
- **React 18.2.0** - 최신 React Hooks 사용
- **React Router 6.16.0** - SPA 라우팅
- **CSS3** - 반응형 디자인, 애니메이션
- **Webpack 5** - 모듈 번들러

### **Backend & Database**
- **Firebase 11.8.1** - 백엔드 서비스
- **Firestore** - NoSQL 실시간 데이터베이스
- **Firebase Auth** - 인증 서비스
- **Security Rules** - 데이터 보안

### **Development**
- **Babel** - ES6+ 트랜스파일링
- **Webpack Dev Server** - 개발 서버
- **ESLint** - 코드 품질 관리

## 🚀 설치 및 실행

### **1. 저장소 클론**
```bash
git clone [repository-url]
cd recipe-sharing-app
```

### **2. 의존성 설치**
```bash
npm install
```

### **3. Firebase 설정**
1. Firebase 콘솔에서 새 프로젝트 생성
2. `src/firebase.js`에 Firebase 구성 정보 입력
3. Firestore 및 Auth 서비스 활성화

### **4. 개발 서버 시작**
```bash
npm start
```

### **5. 프로덕션 빌드**
```bash
npm run build
```

## 📱 앱 구조

```
src/
├── components/          # 재사용 가능한 컴포넌트
│   ├── Header.jsx      # 통일된 상단 헤더
│   └── BottomNav.jsx   # 하단 네비게이션
├── contexts/           # React Context
│   └── AuthContext.jsx # 인증 상태 관리
├── pages/              # 페이지 컴포넌트
│   ├── Home.jsx        # 메인 홈페이지
│   ├── Login.jsx       # 로그인 페이지
│   ├── Register.jsx    # 회원가입 페이지
│   ├── Search.jsx      # 검색 페이지
│   ├── Recipe.jsx      # 레시피 상세 페이지
│   ├── RecipeForm.jsx  # 레시피 작성 페이지
│   ├── MyPage.jsx      # 마이페이지
│   └── Favorites.jsx   # 찜 페이지
├── styles/             # CSS 스타일
│   ├── Header.css      # 헤더 컴포넌트 스타일
│   └── Home.css        # 홈페이지 스타일
└── firebase.js         # Firebase 설정
```
